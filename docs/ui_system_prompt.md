# Veloura Luxury - UI System Design Prompt

Design and implement a complete UI design system for a luxury women’s jewellery brand named “Veloura Luxury”.

The UI system must ensure visual consistency, premium feel, and scalable design across the entire website.

---

## TYPOGRAPHY SYSTEM

Define a clean, high-fashion typography hierarchy:

- **Heading Font:**
  Elegant serif style (e.g., Playfair Display or equivalent)
  Use for:
  - Hero titles
  - Section headings
  - Product names

- **Body Font:**
  Clean modern sans-serif (e.g., Poppins or equivalent)
  Use for:
  - Descriptions
  - UI text
  - Labels

**Rules:**
- Maintain generous line-height (1.5–1.8)
- Avoid excessive bold usage
- Keep typography airy and readable

---

## SPACING SYSTEM

Use a consistent spacing scale across the UI:

- XS: 8px  
- SM: 16px  
- MD: 24px  
- LG: 40px  
- XL: 64px  
- XXL: 96px  

**Rules:**
- Do not use random spacing values
- Maintain consistent padding and margins
- Ensure balanced whitespace

---

## BORDER RADIUS SYSTEM

- Small: 8px  
- Medium: 16px  
- Large: 24px  

**Usage:**
- Cards: 16px  
- Buttons: 12–16px  
- Inputs: 12px  

---

## COLOR SYSTEM

**Primary Background:**
- Ivory: #FFF8F5

**Accent Colors:**
- Blush Pink: #F8D7DA
- Soft Lavender: #E6D6F5
- Blossom Pink: #F2C6CC

**Luxury Accent:**
- Gold: #C9A24E (use sparingly)

**Text:**
- Primary: #2B2B2B
- Secondary: #6E6E6E

**Rules:**
- Maintain soft, pastel luxury feel
- Avoid bright or saturated colors
- Gold only for highlights and key elements

---

## BUTTON SYSTEM

**Primary Button:**
- Background: Gold (#C9A24E)
- Text: White
- Padding: 12px 24px
- Border radius: 14px
- Hover: Slight scale increase, Subtle brightness change

**Secondary Button:**
- Transparent background
- Gold border (1px solid #C9A24E)
- Gold text
- Hover: Smooth fill with gold, Text changes to white

---

## PRODUCT CARD SYSTEM

**Structure:**
- Product image
- Product name
- Price
- Wishlist icon

**Style:**
- Background: White or soft pastel
- Border radius: 16px
- Soft shadow
- Hover: Slight upward lift, Image zoom-in effect

---

## INPUT FIELD SYSTEM

**Style:**
- Background: White
- Border: 1px solid light gray (#E5E5E5)
- Border radius: 12px
- Padding: 12px
- Focus: Border changes to gold, Subtle glow effect

---

## SECTION SYSTEM

- Each section must have:
  - Vertical padding: 64px
  - Clean separation from other sections
  - Centered content alignment
- Maintain strong use of whitespace

---

## SHADOW SYSTEM

**Default:**
- 0 10px 30px rgba(0,0,0,0.05)

**Hover:**
- 0 20px 40px rgba(0,0,0,0.08)

**Rules:**
- Keep shadows soft and subtle
- Avoid heavy or harsh shadows

---

## ANIMATION GUIDELINES

- Include smooth animations across UI components
- Ensure transitions are soft and premium
- Avoid fast or abrupt movements

---

## GENERAL UI RULES

- Maintain minimal design
- Avoid clutter and unnecessary elements
- Ensure consistent styling across all components
- Keep layout clean and balanced

---

## FINAL EXPECTATION

The UI system must feel:
- Premium
- Clean
- Elegant
- Consistent
- Scalable

It should reflect a luxury fashion brand and avoid any appearance of a generic or template-based design.
