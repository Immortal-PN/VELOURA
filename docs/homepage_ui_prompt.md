# Veloura Luxury - Homepage UI Prompt

Design and build the homepage UI for a premium women’s jewellery brand named “Veloura Luxury”.

The homepage must be clean, structured, minimal, and conversion-focused. It should guide users smoothly from discovery to purchase.

---

## GENERAL REQUIREMENTS

- Multi-section homepage layout
- Mobile-first responsive design
- Clean navigation and hierarchy
- Strong use of whitespace
- Consistent UI system (typography, spacing, buttons, cards)

---

## NAVBAR

**Structure:**
- Left: Brand logo
- Center: Navigation links (Home, Shop, Categories, About, Help)
- Right: Icons (Search, Wishlist, Cart, Profile)

**Behavior:**
- Sticky on scroll
- Changes background on scroll for visibility

---

## HERO SECTION

**Layout:**
- Split layout (text on one side, product image on other)

**Content:**
- Main heading: “Luxury That Feels Expensive. Isn’t.”
- Supporting text: short and minimal
- Primary CTA button: “Shop Now”

**Requirements:**
- Large spacing
- Clear visual focus
- Strong hierarchy between heading, text, and button

---

## FEATURED PRODUCTS SECTION

**Structure:**
- Section title
- Grid or row of product cards (4 items visible)

**Each product card must include:**
- Product image
- Product name
- Price
- Wishlist icon

**Behavior:**
- Horizontal scroll on smaller screens

---

## CATEGORIES SECTION

**Structure:**
- Grid or row of category items

**Each category includes:**
- Image or icon
- Category name

**Examples:**
- Rings
- Necklaces
- Bracelets
- Earrings

---

## TRUST SECTION

**Structure:**
- Horizontal layout with 3–4 items

**Each item includes:**
- Icon
- Short label

**Examples:**
- Free Shipping
- Cash on Delivery
- Premium Quality
- Easy Returns

---

## EDITORIAL SECTION

**Structure:**
- Image + text layout

**Content:**
- Heading (brand-focused message)
- Short descriptive paragraph

**Purpose:**
- Build brand identity
- Add storytelling element

---

## BEST SELLERS SECTION

**Structure:**
- Section title
- Product grid (6–8 items)

**Each card follows same product card system**

---

## REVIEWS / SOCIAL PROOF SECTION

**Structure:**
- Multiple review cards

**Each card includes:**
- User name
- Rating
- Short review text

---

## FINAL CTA SECTION

**Structure:**
- Centered layout

**Content:**
- Strong short heading (brand message)
- CTA button (Shop Now)

**Purpose:**
- Drive final conversion

---

## FOOTER

**Structure:**
- Brand logo
- Navigation links
- Help section
- Contact information

---

## UI COMPONENT REQUIREMENTS

- Use consistent spacing across all sections
- Maintain uniform card design
- Keep typography hierarchy clear
- Use consistent button styles

---

## INTERACTIONS

- Include smooth interactions across:
  - Navigation
  - Buttons
  - Product cards
  - Section transitions

---

## PERFORMANCE

- Optimize images
- Ensure fast load time
- Maintain responsiveness across devices

---

## RESTRICTIONS

- Avoid cluttered layouts
- Avoid excessive text
- Avoid inconsistent spacing
- Avoid template-like appearance

---

## FINAL EXPECTATION

The homepage should be:
- Clean and structured
- Easy to navigate
- Visually balanced
- Conversion-focused

It should guide users naturally from browsing to purchasing while maintaining a premium brand feel.
